Linux高级系统运维进阶课程 LVM存储管理/Shell脚本开发/系统性能分析

🔅 10/2/2024

# 课程介绍:

本课程旨在帮助系统运维工程师深入学习基于Ubuntu Linux的高级系统管理技能，并实现技能的进阶。课程内容涵盖了LVM存储管理、Shell脚本开发、系统性能分析等关键领域，以及云计算技术、备份容灾管理、网络配置管理等实际应用场景。通过理论讲解和实际操作演练，学员将全面掌握Ubuntu Linux系统的管理、优化和安全等方面的技能，为应对复杂的系统管理挑战做好准备。

# 课程章节:

LVM存储管理
Shell 脚本语言开发与使用
Ubuntu Server 操作系统基础知识
Ubuntu Server 操作系统管理命令
云计算技术
备份容灾管理
应用服务管理
应用负责均衡服务管理
服务架设管理
用户安全管理
系统安全管理
系统性能分析
系统日志管理
网络基础知识及配置管理
