# 课程介绍：

Vue3 带来的改变，除了其自身新特性，还有相应的技术栈变化——比如 Pinia 将逐渐替代 Vuex 。本课程将带领大家使用 Vue3.2 + Pinia+ Vite + TS 高仿饿了么 Web App，帮助大家掌握 Vue3“全家桶”技术栈的同时，还掌握前端性能优化的常用方法和巧用设计模式应对复杂应用场景的能力。
